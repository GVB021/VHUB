## Introdução aos Agency Agents

Os Agency Agents são personalidades de IA especializadas, cada uma com expertise em uma área específica de desenvolvimento, design, marketing ou gestão de projetos. Eles podem ser ativados para auxiliar em tarefas específicas, fornecendo orientação especializada e soluções práticas.

## Agentes Recomendados por Tarefa

### Desenvolvimento Frontend

- **Frontend Developer**: Para desenvolvimento de componentes React, implementação de UI, otimização de performance e acessibilidade.
- **UI Designer**: Para criação de interfaces consistentes, sistemas de design e especificações visuais.
- **UX Researcher**: Para avaliar a experiência do usuário e propor melhorias baseadas em princípios de UX.

### Desenvolvimento Backend

- **Backend Developer**: Para desenvolvimento de APIs, integração com banco de dados e lógica de negócios.
- **Database Architect**: Para otimização de esquemas de banco de dados, consultas e migrações.
- **DevOps Engineer**: Para configuração de CI/CD, implantação no Railway e monitoramento.

### Áudio e Vídeo

- **Audio Engineer**: Para otimização do módulo de gravação de áudio, processamento de áudio e análise de qualidade.
- **Video Streaming Specialist**: Para implementação e otimização de streaming de vídeo e sincronização.

### Marketing e Conteúdo

- **Content Creator**: Para criação de conteúdo para minicursos, descrições e materiais educacionais.
- **Growth Marketer**: Para estratégias de aquisição de usuários e otimização de conversão.

### Gestão de Projeto

- **Project Manager**: Para planejamento de sprints, acompanhamento de tarefas e gestão de recursos.
- **Product Manager**: Para definição de roadmap, priorização de features e análise de métricas.

## Como Ativar um Agente

Para ativar um agente, você pode usar o Claude AI ou outro LLM compatível e solicitar a ativação do agente específico:

```
Por favor, ative o agente Frontend Developer para me ajudar a implementar um novo componente React para a página de minicursos.
```

Ou para tarefas mais específicas:

```
Por favor, ative o agente Audio Engineer para me ajudar a otimizar o módulo de gravação de áudio do UltimoHub.
```

## Exemplos de Uso

### Exemplo 1: Otimização de Performance Frontend

```
Ative o agente Frontend Developer para analisar e otimizar a performance do componente CourseGrid.tsx que está renderizando lentamente com muitos cursos.
```

### Exemplo 2: Melhoria na Qualidade de Áudio

```
Ative o agente Audio Engineer para implementar um algoritmo de redução de ruído no módulo de gravação de áudio.
```

### Exemplo 3: Criação de Conteúdo para Minicursos

```
Ative o agente Content Creator para desenvolver a estrutura e conteúdo de um novo minicurso sobre técnicas avançadas de dublagem.
```

### Exemplo 4: Otimização de Banco de Dados

```
Ative o agente Database Architect para otimizar as consultas ao banco de dados que estão causando lentidão na listagem de minicursos.
```

## Combinando Agentes para Tarefas Complexas

Para tarefas que envolvem múltiplas áreas de expertise, você pode solicitar a combinação de agentes:

```
Preciso implementar um novo recurso de feedback em tempo real durante as sessões de gravação. Por favor, combine os agentes Frontend Developer, Audio Engineer e UX Researcher para me ajudar com esta tarefa.
```

### Fluxo de Trabalho Recomendado

1. **Análise**: Use o UX Researcher para entender as necessidades dos usuários e definir os requisitos.
2. **Design**: Use o UI Designer para criar o design visual e as especificações.
3. **Implementação**: Use o Frontend Developer ou Backend Developer para implementar a solução.
4. **Otimização**: Use especialistas técnicos (Audio Engineer, Database Architect) para otimizar aspectos específicos.
5. **Teste e Validação**: Use o QA Specialist para testar e validar a implementação.

---

Lembre-se de que os agentes são ferramentas de assistência e suas recomendações devem ser avaliadas criticamente antes da implementação. Combine o conhecimento especializado dos agentes com seu próprio entendimento do projeto para obter os melhores resultados.
